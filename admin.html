¿Cómo preparo el Excel?

Poné estas columnas en la primera fila: nombre, categoria, tipo, expansion, precio, estado, etiqueta, cantidad, imagen. Llená una fila por carta. En Excel/Google Sheets: Archivo → Descargar → CSV. La columna imagen es un link directo a la foto (que termine en .jpg/.png/.webp) — la tienda la carga sola, no hace falta subir nada. Si la dejás vacía, la carta sale con emoji y le podés agregar la foto después desde la izquierda. Ojo: el link tiene que ser público y de un sitio que permita mostrar la imagen en otra página.

¿Cómo funciona / dónde saco el token?

Al publicar, tu inventario se sube a GitHub y la tienda (en Vercel) se actualiza sola en ~1 min. El token va a github.com/settings/tokens → token fine-grained con permiso Contents: Read and write sobre el repo reroll-hobby-store. Se guarda solo en este navegador; lo pegás una vez.
