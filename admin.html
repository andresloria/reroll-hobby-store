¿Cómo preparo el Excel?

Poné estas columnas en la primera fila: nombre, categoria, precio, estado, etiqueta. Llená una fila por carta. En Excel/Google Sheets: Archivo → Descargar → CSV. Las fotos las agregás luego una por una desde el panel de la izquierda (o las dejás con emoji por defecto).

¿Cómo funciona / dónde saco el token?

Al publicar, tu inventario se sube a GitHub y la tienda (en Vercel) se actualiza sola en ~1 min. El token va a github.com/settings/tokens → token fine-grained con permiso Contents: Read and write sobre el repo reroll-hobby-store. Se guarda solo en este navegador; lo pegás una vez.
